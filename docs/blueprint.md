# **App Name**: LifeTrack

## Core Features:

- Learning Progress Tracker: Track learning progress by skill, difficulty, and daily targets.  Includes daily and monthly reports with streaks and missed days.
- Smart Budget System: Manage monthly budget, fixed expenses, and daily rolling budget with carry forward, weekend bonuses, and holiday bonuses.
- Expense Tracker: Record and categorize daily expenses.  Automatically recalculate daily and monthly budgets upon expense updates or deletions.
- Automated Insights Tool: Generative AI tool providing insights into spending habits, such as overspending days and category-wise spending analysis. This feature helps users understand their financial behavior based on the expense data tool.
- Daily Diary: Create private daily diary entries with fields for daily activities, learnings, challenges, and plans.
- Data Persistence: Use Firestore database for data persistence.
- User Authentication: Secure user authentication with email and password, integrated with Firebase Authentication.  Ensures data privacy by linking all data to specific user IDs.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to evoke sophistication and organization.
- Background color: Very light purple (#F2F0F9), near white.
- Accent color: Blue (#5A87D1) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral, readable aesthetic.
- Use consistent line-style icons for learning, budget, and diary modules.
- Dashboard layout with clean cards and tables for easy data visualization. Prioritize mobile responsiveness.
- Subtle transition animations for navigation and data updates.
